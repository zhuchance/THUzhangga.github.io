Win Server 2012 R2安装Oracle

在官网上下好：winx64_12201_database.zip

解压后点击setup.exe，突然dos窗口一闪而过。里面的信息大概是显示器256种颜色检测通过，之后就没有了。到：C:\Users\Administrator\AppData\Local\Temp底下查看安装日志：

点进一个如OraInstall2018-05-15_01-02-01PM的文件夹，里面有安装日志：installActions2018-05-15_01-02-01PM.log，内部信息为：

可见关键问题在最后两行。一番google之后发现：

https://community.oracle.com/thread/2393033

编号为938305这个老哥从3月18号提出这个问题，过了一周才被自己解决，真得感谢他。HDD太大了，压缩卷即可：

Cmd中输入diskmgmt.msc，在磁盘空白处右键压缩卷，然后对未分配右键新建简单卷即可。我让C盘剩了700多G后也能安装。下面是一些一些安装信息:

系统类选择桌面类

Oracle主目录用户选择现有Windows用户(ga秘密为P@ssw0rd)

典型安装部分更改字符集为ZHS16GBK

Oracle口令为xxxxxxxxx
