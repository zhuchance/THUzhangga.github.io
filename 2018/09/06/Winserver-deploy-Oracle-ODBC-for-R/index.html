Winserver deploy Oracle ODBC for R

Posted on 2018-09-06

前言

学了一点R后深深迷恋attach功能，毕竟Python里每加载一个变量都要dataframe['var']

R及RODBC离线安装

当前最新版本为R-3.5.1。

由于Winser无网络连接，需要掌握离线安装package的姿势：

用一段程序check dependencies并且下载好。

经过check发现ROCBC好像依赖stats，但是又找不到这个包…于是直接到https://cran.r-project.org/web/packages/RODBC/index.html 下载windows binaries，选择r-release就好。

在R或者R studio都可以install package from local files. 可是安装时发现出错。原来是RODBC只支持到R-3.4.1。于是重新下载安装。折腾一阵后终于成功！

配置Oracle ODBC

安装完后美滋滋地测试连接：

但却报错：

原来是没有配置Oracle ODBC。

打开控制面板\系统和安全\管理工具\ODBC 数据源(64 位)。如下图：

切换到“系统DSN”，点击添加，选择Oracle in OraDB12Home1：

然后配置下：

上图中Data Source Name、Descrpiton、TNS Service Name我都填的数据库名，User ID我填的oracle用户名。然后Test Connection，出现一个框让填写密码：

填写后点击OK，如果出现下图则连接成功。最后点击OK即可添加。

最后就可以愉快地开始用R做分析了！
