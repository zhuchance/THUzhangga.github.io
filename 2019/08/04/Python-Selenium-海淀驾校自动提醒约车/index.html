Python Selenium 海淀驾校自动提升约车

Posted on 2019-08-04

前言

暑假报名了海淀驾校的散班，需要自行约科目二练习。在官方APP上，每天早上7:35开始开放部分名额，但是起了好几次早愣是没抢到。室友告诉我需要自己有时间就刷APP，兴许能捡漏。但我试了好久每次都是0车可约。于是我开始想能不能像12306抢票一样自动在后台不停地刷，一旦有人退立刻能抢到。

约车官网分析

海驾约车官网是http://haijia.bjxueche.net/

打开以后界面如下：

底下的温馨提示没有截全，完整版应该是这样的：

看来早就有人干过刷车了，还是小心为上。

从约车界面分析需要填写用户名、密码和验证码，检查元素发现它们的id分别是txtUserName，txtPassword，txtIMGCode。一开始我试着使用过requests进行爬取：

结果发现验证码的url很奇怪：

可见它每次都会调用Math.random()生成一个随机数构造出验证码的url。

难道我们就束手无力了？不，Selenium还能截图！

Selenium验证码截取

Selenium是个web自动化测试工具，模仿浏览器行为爬虫，但是效率较低，一开始我并没有考虑。考虑到我的需求，约车的人应该不会特别多，慢点也没关系。

用Selenium打开网站后，发现窗口太小看不到验证码，于是要先滑过去。然后保存截图。

保存截图后用PIL工具将验证码截取出来，这里还需要知道验证码在截图中的位置，可以用画图工具分析。下图显示了验证码左上角像素位置。

Tessract验证码识别

Tessract一款由HP实验室开发由Google维护的开源OCR（Optical Character Recognition，光学字符识别）引擎。安装方式可以参考：tesseract 安装及使用

安装完成后即可对验证码识别：

从实际使用体验来看，Tessract成功的概率较低，需要重复几次。同时验证码错误时需要关闭警告，否则程序会报错自动退出：
