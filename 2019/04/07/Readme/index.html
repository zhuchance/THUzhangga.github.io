家庭背景

香港回归的那个夏天，我出生在湖南平江的一个偏僻小山村。我的祖父母、父母都是农民，虽然文化程度和收入都不高，但是他们如广大中国农民一般淳朴、善良、勤劳，我或多或少也继承了一些他们的特点。我的童年过的很愉快，没有补习班和艺术班，整天上山摘果，下河摸鱼。

学习经历

初中寄宿周末回家，需要骑10公里的山路自行车。高中寄宿月末回家，需要坐1小时大巴再走10公里山路。整个中学阶段在努力学习中度过，一所普通的贫困县一中水平并不高，即使考第一而且高第二几十分，也难考入清华。所幸获得了自主招生资格，2014年1月顺利通过清华自强计划初审，到北京参加面试，之后通过面试获得50分降分。2014年高考发挥失常，650分距离清华录取线尚有20多分距离，凭降分考入清华水利系。

大学阶段由于工科学习压力大，一直以保研为目标拼搏。期间在学生会、学生科协打过杂，但总之不是很喜欢做社工。

参加过一些项目/比赛：

  • 2017以及2018美赛H奖（菜鸡奖）
  • 闯世界计划（201807），美帝Hopkins两月打杂，然而没出什么成果。
  • EGU (201904)，5人问津的poster展示

目前在清华水利系河流研究所读博，研究方向为数字流域模型。
